[Kotlin]Constructor

2018-08-16

This 키워드는 클래스 이름을 나타내는 대명사라고 생각하면된다.

  • Field : 데이터가 저장되는 공간
  • Getter: 저장된 값을 읽을때 호출되는 함수
  • Setter : 값을 저장하려고 할 떄 호출되는 함수.

TIP)
val 프로피터는 초기 값이 주어지면 더 이상 값을 변경(set)할 수 없다.
그래서 val 프로피터에는 Getter만 존재한다.
val 프로피터에 Setter을 정의하려고 하면 오류 발생 !

var age =0 / get / set
